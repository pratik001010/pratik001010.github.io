About Me

Erasmus Mundus MSc student in Embedded Intelligence and IoT Systems, passionate about transforming loT technologies into innovative digital solutions.

My background combines embedded hardware experience with an interest in loT-driven business models, smart services, and data analytics.

Experienced in STM32, ESP32, LoRaWAN, and Zigbee, I focus on connecting devices to cloud platforms for automation and insight generation. I am particularly interested in how loT can support digital innovation, sustainability, and the development of connected business ecosystems.

Featured Projects

LoRaWAN Indoor Path Loss Analysis

Investigates how environmental conditions affect LoRaWAN signal reliability in offices. Uses ML models (MLR, RF, XGBoost) on RSSI/SNR data to predict path loss and optimize spreading factors.

View on GitHub
Intelligent Home Automation System

Integrated system on Arduino Mega with biometric/RFID access, ESP32-CAM surveillance, and gas/fire detection. Features multi-mode control via IR remote and offline voice recognition.

View on GitHub
HEXAPOD ROBOT

A six-legged robot with 18 degrees of freedom (3 servos per leg). Focuses on gait generation, balance control, and terrain adaptability using inverse kinematics algorithms on an Arduino platform.
